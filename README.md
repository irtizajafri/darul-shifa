# Durul Shifa Imam khamenei  Hospital Management System

Multi-Module Dashboard with HR & Payroll Phase 1

## Tech Stack
- React.js + Vite
- JavaScript
- Tailwind CSS + SCSS
- Lucide React (icons)
- Recharts, TanStack Table
- React Router v6
- Zustand (auth + module state)
- React Hook Form
- Axios (mock)
- react-hot-toast

## Getting Started

```bash
npm install
npm run dev
```

Demo login: `admin@hospital.com` / `admin123`

## Project Structure
- `/src/components/ui` - Button, Input, Modal, Badge, Card, etc.
- `/src/components/layout` - Sidebar, Navbar, MainLayout
- `/src/pages` - Auth, Dashboard, Employees, Attendance, GatePass, ShortLeave, Advance, Reports
- `/src/store` - useAuthStore, useModuleStore
- `/src/utils` - dummyData, helpers
